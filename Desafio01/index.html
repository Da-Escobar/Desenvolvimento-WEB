HTML

HTML é uma linguagem de marcação, utilizada para montar a estrutura do seu site, seguido sempre as regras semânticas. Para classificar cada elemento utilizamos as tags, que ficam dentro do símbolo de menor e maior.
Exemplo: < h >

css

CSS são as folhas de estilização, usado para desenvolver a aparência e estilização do site. Para selecionar o que deve ser estilizado, utilizamos os seletores. Eles podem ser através de tags, class ou id.
